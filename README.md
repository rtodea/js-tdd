# js-tdd
Starter kit for trying TDD

## Installation

    npm install

## Unit test tech stack

1. [MochaJS](mochajs.org)
1. [Chai](chaijs.com)
1. [SinonJS](sinonjs.org)

### Running tests

    npm run test

## Code linter

[Airbnb ESLint Specifications](https://github.com/airbnb/javascript)

### Running linter

    npm run lint
